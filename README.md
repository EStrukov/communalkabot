# Телеграмм бот для напоминания коммуналки

Бот автоматически напоминает о необходимости скинуть коммуналку 20 числа каждого месяца и ежедневно, если не скинули.

## Команды

- `/communal` - отправить фото коммуналки
- `/pay` - отправить фото/ссылку на чек оплаты
- `/status` - проверить статус

## Установка

1. Создайте бота в Telegram через @BotFather и получите токен
2. Замените `your_telegram_bot_token_here` в файле `.env` на ваш токен
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите бота:
   ```bash
   node bot.js
   ```

## Функционал

- Автоматические напоминания 20 числа каждого месяца
- Ежедневные напоминания если не скинули коммуналку
- Сохранение фото коммуналки
- Сохранение чеков оплаты
- Просмотр статуса
